Welcome to See Click Fixit for Hampton Roads.

Getting Started


$ git clone git@github.com:troy0820/fixit.git
$ cd fixit
$ npm install
$ npm install --save bower
$ bower install bootstrap
$ bower install font-awesome
$ bower install leaflet.markercluster

The purpose of this project is to annotate the issues in the Hampton Roads area on a map to generalize the issues per city.
