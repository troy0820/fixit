Welcome to See Click Fixit for Hampton Roads.

Getting Started


$ git clone git@github.com:troy0820/fixit.git
$ cd fixit
$ npm install
$ npm install --save bower
$ bower install bootstrap
$ bower install font-awesome
$ bower install leaflet.markercluster
$ npm start 
Open Browser to http://localhost:3000

The purpose of this project is to annotate the issues in the Hampton Roads area on a map to generalize the issues per city. Currently Virginia Beach is the only city who utilizes the service. This is intended to show the issues within the 7 cities.

See Click Fix it api

See Click Fix it api for Hampton Va
The seven cities in this project include:

  • Hampton
  • Norfolk
  • Newport News
  • Portsmouth
  • Chesapeake
  • Suffolk
  • Virginia Beach
