Welcome to See Click Fixit for Hampton Roads.

Getting Started


 Make sure you have node installed

$ git clone git@github.com:troy0820/fixit.git
$ cd fixit
$ npm install
$ npm start 

Open Browser to http://localhost:3000

The purpose of this project is to annotate the issues in the Hampton Roads area on a map to generalize the issues per city.

See Click Fix it api

See Click Fix it api for the city of Hampton
The Hampton Roads cities in this project include:

  • Hampton
  • Norfolk
  • Newport News
  • Portsmouth
  • Chesapeake
  • Suffolk
  • Virginia Beach
